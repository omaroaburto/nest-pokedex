# Ejecutar en desarrollo

1. Clonar el repositorio
2. Instalar dependencias
```
yarn install
```
3. Instalar Nest CLI
```
npm i -g @nestjs/cli
```
4. Levantar base de datos
```
docker-compose up -d
``` 

## Stack usado
* NestJs
* MongoDB
